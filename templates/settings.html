Verification code acquisition strategy

  • Progressive checking: only unread messages are checked in the first 3 polls, then all messages are checked
  • Timestamp filtering: Automatically skip old emails before OTP is sent
  • Verification code deduplication: Avoid reusing the same verification code
  • Multi-strategy extraction: topic priority → semantic matching → covert matching
  • Sender Verification: Strictly verify that the email comes from the official OpenAI
